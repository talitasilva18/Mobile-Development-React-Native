Mega Store – Aplicativo Mobile

Este projeto consiste no desenvolvimento de um aplicativo mobile utilizando React Native com Expo. O aplicativo permite visualizar produtos de diferentes categorias utilizando dados obtidos através da API pública DummyJSON.

O objetivo do aplicativo é demonstrar conceitos básicos de desenvolvimento mobile, consumo de API, navegação entre telas e organização do código.

_________________________________________________________________________________________________________________________________

Tecnologias utilizadas

React Native

Expo

Axios

React Navigation

API DummyJSON

_________________________________________________________________________________________________________________________________

Como executar o projeto:

Instalar as dependências:

npm install

Iniciar o projeto:

npx expo start

Abrir o aplicativo utilizando o Expo Go no celular ou no navegador.
